# SweetDreams

SweetDreams is a web application which gradually lowers the volume of YouTube videos to 0 over a set period of time. This application is perfect for users who enjoy falling asleep with noise but don't want to fiddle around with the volume of their devices. You can use SweetDreams anytime at https://jesseblacklock.com/SweetDreams.php.

---

## How Do I Use SweetDreams?

1. Paste the URL of a YouTube video in the input field 
2. Designate when you want the volume to reach 0. You can do this by either setting a target time, or by selecting the 5 minute, 10 minute, 15 minute, etc. options.
3. Sweet dreams!

---

## Features

The latest iteration of SweetDreams also features a blackout mode option. This darkens the screen to prevent the light from your device's screen from disturbing your sleep. Select the Blackout Mode button to activate Blackout Mode. To exit Blackout Mode, simply presse the Exit Blackout button or press escape.

---

## Supported Operating Systems

Currently SweetDreams works on all operating systems except macOS and iOS.
